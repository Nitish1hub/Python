## String Basics Recap

**Define: String** - A sequence of characters enclosed in quotes.

**Define: Immutable** - Strings cannot be changed after creation; operations create new strings.

### Creating Strings

```python
# Single quotes
message = 'Hello'

# Double quotes
message = "Hello"

# Both are identical!
print('Hello' == "Hello")  # True
```

### Basic String Operations

```python
# Concatenation
greeting = "Hello" + " " + "World"
print(greeting)  # Hello World

# Repetition
repeated = "Ha" * 3
print(repeated)  # HaHaHa

# Length
text = "Python"
print(len(text))  # 6

# Indexing (zero-based)
text = "Python"
print(text[0])   # P
print(text[-1])  # n

# Slicing
text = "Python"
print(text[0:3])   # Pyt
print(text[2:])    # thon
```

### Strings Are Immutable

```python
text = "Hello"
# text[0] = "h"  # Error! Cannot change strings

# Must create new string
text = "h" + text[1:]
print(text)  # hello
```

## String Methods - Transformation

### upper() and lower()

```python
text = "Hello World"

print(text.upper())  # HELLO WORLD
print(text.lower())  # hello world

# Original unchanged!
print(text)  # Hello World
```

### capitalize() and title()

```python
text = "hello world"

print(text.capitalize())  # Hello world
print(text.title())       # Hello World

# Difference:
sentence = "hello WORLD python"
print(sentence.capitalize())  # Hello world python
print(sentence.title())       # Hello World Python
```

### swapcase()

```python
text = "Hello World"
print(text.swapcase())  # hELLO wORLD
```

### strip(), lstrip(), rstrip()

**Define: Whitespace** - Spaces, tabs, newlines that appear empty.

```python
text = "   Hello World   "

print(text.strip())   # "Hello World" (both ends)
print(text.lstrip())  # "Hello World   " (left only)
print(text.rstrip())  # "   Hello World" (right only)

# Remove specific characters
text = "***Hello***"
print(text.strip("*"))  # Hello
```

### replace()

```python
text = "Hello World"

new_text = text.replace("World", "Python")
print(new_text)  # Hello Python

# Replace all occurrences
text = "one one one"
print(text.replace("one", "two"))  # two two two

# Limit replacements
text = "one one one"
print(text.replace("one", "two", 2))  # two two one
```

### split() - Convert to List

```python
text = "Hello World Python"

words = text.split()  # Split on whitespace
print(words)  # ['Hello', 'World', 'Python']

# Split on specific character
csv = "apple,banana,cherry"
fruits = csv.split(",")
print(fruits)  # ['apple', 'banana', 'cherry']

# Limit splits
text = "one-two-three-four"
print(text.split("-", 2))  # ['one', 'two', 'three-four']
```

### join() - Combine List into String

```python
words = ["Hello", "World", "Python"]

# Join with space
sentence = " ".join(words)
print(sentence)  # Hello World Python

# Join with comma
csv = ",".join(words)
print(csv)  # Hello,World,Python

# Join with nothing
text = "".join(words)
print(text)  # HelloWorldPython
```

## String Methods - Searching

### find() and index()

```python
text = "Hello World"

# find() - returns index or -1 if not found
pos = text.find("World")
print(pos)  # 6

pos = text.find("Python")
print(pos)  # -1 (not found)

# index() - returns index or raises error
pos = text.index("World")
print(pos)  # 6

# pos = text.index("Python")  # Error! ValueError
```

### count()

```python
text = "Hello World, Hello Python"

count = text.count("Hello")
print(count)  # 2

count = text.count("l")
print(count)  # 3

count = text.count("Java")
print(count)  # 0
```

### startswith() and endswith()

```python
text = "Hello World"

print(text.startswith("Hello"))  # True
print(text.startswith("World"))  # False

print(text.endswith("World"))    # True
print(text.endswith("Hello"))    # False

# Case sensitive!
print(text.startswith("hello"))  # False
```

### in Operator

```python
text = "Hello World"

print("World" in text)    # True
print("Python" in text)   # False
print("Hello" not in text)  # False
```

## String Methods - Validation

### isalpha() - Only Letters

```python
print("Hello".isalpha())      # True
print("Hello123".isalpha())   # False
print("Hello World".isalpha()) # False (space!)
```

### isdigit() - Only Digits

```python
print("12345".isdigit())   # True
print("123.45".isdigit())  # False (decimal point)
print("123a".isdigit())    # False
```

### isalnum() - Letters and Digits

```python
print("Hello123".isalnum())   # True
print("Hello 123".isalnum())  # False (space)
print("Hello-123".isalnum())  # False (hyphen)
```

### isspace() - Only Whitespace

```python
print("   ".isspace())     # True
print("  a  ".isspace())   # False
print("".isspace())        # False (empty)
```

### isupper() and islower()

```python
print("HELLO".isupper())   # True
print("Hello".isupper())   # False

print("hello".islower())   # True
print("Hello".islower())   # False
```

## String Formatting

### Old Style - % Formatting

```python
name = "Alice"
age = 25

# %s for string, %d for integer
message = "Name: %s, Age: %d" % (name, age)
print(message)  # Name: Alice, Age: 25

# %f for float
price = 19.99
print("Price: $%.2f" % price)  # Price: $19.99
```

### format() Method

```python
name = "Alice"
age = 25

# Positional arguments
message = "Name: {}, Age: {}".format(name, age)
print(message)  # Name: Alice, Age: 25

# Named arguments
message = "Name: {n}, Age: {a}".format(n=name, a=age)
print(message)  # Name: Alice, Age: 25

# Formatting numbers
price = 19.99
print("Price: ${:.2f}".format(price))  # Price: $19.99
```

### f-strings (Best! Python 3.6+)

**Define: f-string** - Formatted string literal that allows expressions inside curly braces.

```python
name = "Alice"
age = 25

# Simple f-string
message = f"Name: {name}, Age: {age}"
print(message)  # Name: Alice, Age: 25

# Expressions inside
x = 10
y = 20
print(f"{x} + {y} = {x + y}")  # 10 + 20 = 30

# Format numbers
price = 19.99
print(f"Price: ${price:.2f}")  # Price: $19.99

# Call functions
name = "alice"
print(f"Uppercase: {name.upper()}")  # Uppercase: ALICE
```

### Advanced f-string Formatting

```python
# Alignment and width
name = "Alice"
print(f"|{name:>10}|")  # |     Alice| (right align, width 10)
print(f"|{name:<10}|")  # |Alice     | (left align)
print(f"|{name:^10}|")  # |  Alice   | (center)

# Numbers
number = 42
print(f"{number:05d}")  # 00042 (5 digits, pad with zeros)

# Percentage
ratio = 0.755
print(f"{ratio:.1%}")  # 75.5%

# Thousands separator
big_number = 1000000
print(f"{big_number:,}")  # 1,000,000
```

## Escape Characters

**Define: Escape character** - Special character sequences starting with backslash `\`.

### Common Escape Characters

```python

# Output:
# Hello
# World

print('It\'s a nice day')      # It's a nice day
print("She said \"Hello\"")    # She said "Hello"

# Alternative - use different quotes
print("It's a nice day")       # It's a nice day
print('She said "Hello"')      # She said "Hello"
```

### Escape Sequence Reference


## Raw Strings

**Define: Raw string** - String that treats backslashes literally (prefix with `r`).

```python

# Useful for Windows paths
windows_path = r"C:\Users\Alice\Documents\file.txt"
print(windows_path)

# Useful for regex patterns (we'll learn later)
```

## String Manipulation Techniques

### Reversing a String

```python
text = "Python"

# Method 1: Slicing
reversed_text = text[::-1]
print(reversed_text)  # nohtyP

# Method 2: reversed() + join()
reversed_text = "".join(reversed(text))
print(reversed_text)  # nohtyP
```

### Checking Palindrome

```python
def is_palindrome(text):
    # Remove spaces and convert to lowercase
    text = text.replace(" ", "").lower()
    return text == text[::-1]

print(is_palindrome("racecar"))      # True
print(is_palindrome("hello"))        # False
print(is_palindrome("A man a plan a canal Panama"))  # True (ignoring spaces/case)
```

### Removing Characters

```python
text = "Hello, World!"

no_punctuation = text.replace(",", "").replace("!", "")

cleaned = ""
for char in text:
    if char.isalnum() or char.isspace():
        cleaned = cleaned + char
```

### Counting Words

## Real-Life Practical Examples

### Example 1: Email Validator

```python
def is_valid_email(email):
    # Simple validation
    email = email.strip().lower()
    
    # Must contain @
    if "@" not in email:
        return False
    
    # Split by @
    parts = email.split("@")
    if len(parts) != 2:
        return False
    
    username, domain = parts
    
    # Username and domain must not be empty
    if len(username) == 0 or len(domain) == 0:
        return False
    
    # Domain must have a dot
    if "." not in domain:
        return False
    
    return True

# Test
emails = [
    "alice@email.com",
    "bob@",
    "@domain.com",
    "invalid.email",
    "charlie@company.co.uk"
]

for email in emails:
    valid = is_valid_email(email)
    print(f"{email}: {valid}")

# Output:
# alice@email.com: True
# bob@: False
# @domain.com: False
# invalid.email: False
# charlie@company.co.uk: True
```

### Example 4: Password Strength Checker

```python
def check_password_strength(password):
    # Minimum length
    if len(password) < 8:
        return "Weak: Too short (minimum 8 characters)"
    
    # Check for uppercase
    has_upper = False
    for char in password:
        if char.isupper():
            has_upper = True
            break
    
    if not has_upper:
        return "Weak: No uppercase letter"
    
    # Check for lowercase
    has_lower = False
    for char in password:
        if char.islower():
            has_lower = True
            break
    
    if not has_lower:
        return "Weak: No lowercase letter"
    
    # Check for digit
    has_digit = False
    for char in password:
        if char.isdigit():
            has_digit = True
            break
    
    if not has_digit:
        return "Medium: No number"
    
    # Check for special character
    special_chars = "!@#$%^&*()_+-=[]{}|;:,.<>?"
    has_special = False
    for char in password:
        if char in special_chars:
            has_special = True
            break
    
    if not has_special:
        return "Medium: No special character"
    
    return "Strong"

# Test
passwords = [
    "pass",
    "password",
    "Password",
    "Password1",
    "Password1!"
]

for pwd in passwords:
    strength = check_password_strength(pwd)
    print(f"{pwd}: {strength}")

# Output:
# pass: Weak: Too short (minimum 8 characters)
# password: Weak: No uppercase letter
# Password: Weak: No number
# Password1: Medium: No special character
# Password1!: Strong
```

### Example 5: Text Censoring

```python
def censor_text(text, bad_words):
    # Convert to lowercase for checking
    words = text.split()
    censored_words = []
    
    for word in words:
        # Remove punctuation for comparison
        clean_word = ""
        for char in word:
            if char.isalnum():
                clean_word = clean_word + char
        
        # Check if bad word
        if clean_word.lower() in bad_words:
            # Replace with asterisks
            censored = "*" * len(word)
            censored_words.append(censored)
        else:
            censored_words.append(word)
    
    return " ".join(censored_words)

# Test
bad_words = {"bad", "ugly", "hate"}
text = "This is a bad example with ugly and hate words."

censored = censor_text(text, bad_words)
print(censored)
# Output: This is a *** example with **** and **** words.
```

### Example 6: URL Slug Generator

```python
def create_slug(title):
    # Convert to lowercase
    slug = title.lower()
    
    # Replace spaces with hyphens
    slug = slug.replace(" ", "-")
    
    # Remove special characters (keep only alphanumeric and hyphens)
    clean_slug = ""
    for char in slug:
        if char.isalnum() or char == "-":
            clean_slug = clean_slug + char
    
    # Remove multiple consecutive hyphens
    while "--" in clean_slug:
        clean_slug = clean_slug.replace("--", "-")
    
    # Remove leading/trailing hyphens
    clean_slug = clean_slug.strip("-")
    
    return clean_slug

# Test

    "Understanding Data Structures  --  Part 1"
```

## Common Mistakes

### Mistake 1: Trying to Modify Strings

**‚ùå Wrong Way:**
```python
text = "Hello"
# text[0] = "h"  # Error! Strings are immutable
```

**‚úÖ Correct Way:**
```python
text = "Hello"
text = "h" + text[1:]
print(text)  # hello
```

### Mistake 2: Forgetting Strings Are Immutable

**‚ùå Wrong Way:**
```python
text = "hello"
text.upper()  # This doesn't change text!
print(text)  # still "hello"
```

**‚úÖ Correct Way:**
```python
text = "hello"
text = text.upper()  # Assign the result
print(text)  # HELLO
```

### Mistake 3: Off-by-One in Slicing

**‚ùå Wrong Way:**
```python
text = "Python"
# Want "Pyt"
print(text[0:4])  # Pyth  ‚Üê Too much!
```

**‚úÖ Correct Way:**
```python
text = "Python"
print(text[0:3])  # Pyt  ‚Üê Correct
```

### Mistake 4: Case-Sensitive Comparisons

**‚ùå Wrong Way:**
```python
user_input = "YES"
if user_input == "yes":  # False!
    print("Confirmed")
```

**‚úÖ Correct Way:**
```python
user_input = "YES"
if user_input.lower() == "yes":  # True
    print("Confirmed")
```

### Mistake 5: Not Stripping Whitespace

**‚ùå Wrong Way:**
```python
user_input = "  alice@email.com  "
if user_input == "alice@email.com":  # False (extra spaces)
    print("Valid")
```

**‚úÖ Correct Way:**
```python
user_input = "  alice@email.com  "
if user_input.strip() == "alice@email.com":  # True
    print("Valid")
```

## Best Practices

### ‚úÖ Do This:

1. **Use f-strings for Formatting (Modern Python)**
   ```python
   # Good - clear and concise
   name = "Alice"
   age = 25
   print(f"{name} is {age} years old")
   ```

2. **Use join() for Building Strings in Loops**
   ```python
   # Good - efficient
   words = ["Hello", "World", "Python"]
   sentence = " ".join(words)
   
   # Avoid - slow for many iterations
   sentence = ""
   for word in words:
       sentence = sentence + word + " "
   ```

3. **Normalize Input Data**
   ```python
   # Good - handle variations
   email = user_input.strip().lower()
   ```

4. **Use String Methods**
   ```python
   # Good - use built-in methods
   if text.startswith("http"):
       process_url(text)
   
   # Avoid - manual checking
   if text[0:4] == "http":
       process_url(text)
   ```

5. **Use Raw Strings for Paths**
   ```python
   # Good - no escape issues
   path = r"C:\Users\Alice\Documents"
   ```

### ‚ùå Avoid This:

1. **Don't Concatenate in Loops**
   ```python
   # Avoid - creates many intermediate strings
   result = ""
   for i in range(1000):
       result = result + str(i)
   
   # Better
   result = "".join(str(i) for i in range(1000))
   ```

2. **Don't Forget to Assign Results**
   ```python
   # Avoid
   text = "hello"
   text.upper()  # Doesn't change text!
   
   # Correct
   text = text.upper()
   ```

3. **Don't Use + for Formatting**
   ```python
   # Avoid - hard to read
   message = "Hello " + name + ", you are " + str(age) + " years old"
   
   # Better
   message = f"Hello {name}, you are {age} years old"
   ```

## Summary

Congratulations! You've mastered advanced string manipulation in Python! üéâ

**üéØ Key Takeaways:**

**üìå String Properties**
- Immutable - can't change after creation
- Indexed from 0
- Sliceable with `[start:end]`
- Methods return new strings

**üìå Transformation Methods**
- `upper()`, `lower()` - change case
- `strip()`, `lstrip()`, `rstrip()` - remove whitespace
- `replace()` - replace substrings
- `split()` - string to list
- `join()` - list to string

**üìå Searching Methods**
- `find()`, `index()` - locate substrings
- `count()` - count occurrences
- `startswith()`, `endswith()` - check beginning/end
- `in` operator - check membership

**üìå Validation Methods**
- `isalpha()` - only letters
- `isdigit()` - only digits
- `isalnum()` - letters and digits
- `isspace()` - only whitespace
- `isupper()`, `islower()` - check case

**üìå String Formatting**
- **f-strings** (best): `f"{name} is {age}"`
- `format()`: `"{} is {}".format(name, age)`
- `%` formatting: `"%s is %d" % (name, age)`

**üìå Special Strings**
- Escape characters: `\n`, `\t`, `\\`, `\'`, `\"`
- Raw strings: `r"C:\path\to\file"`
- Multi-line: `"""text"""`

**üí° Remember:**
- Strings are immutable - assign results!
- Use f-strings for formatting
- Use `join()` for building strings
- Normalize user input (strip, lower)
- Use built-in methods
- Case matters (use `.lower()` for comparisons)
