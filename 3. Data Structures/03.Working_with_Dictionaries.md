## Introduction - Looking Up Data by Name

Imagine storing student information in a list:

```python
student = ["Alice", 25, "Computer Science", 3.8]
```

Which element is the name? The age? You have to remember positions! And what if you want to look up a student's grade by their name? With lists, you'd need to search through every element.

**There's a better way: Dictionaries!**

```python
student = {
    "name": "Alice",
    "age": 25,
    "major": "Computer Science",
    "gpa": 3.8
}

# Now you can access by meaningful names!
print(student["name"])  # Alice
print(student["gpa"])   # 3.8
```

**Real-world examples where dictionaries excel:**
- Phone book (name ‚Üí phone number)
- User profiles (username ‚Üí user data)
- Configuration settings (setting name ‚Üí value)
- Shopping cart (product ‚Üí quantity)
- Language translation (English word ‚Üí Spanish word)

**In this blog, you'll learn:**
- How to map keys to values
- Creating and accessing dictionaries
- Adding, updating, and removing entries
- Looping through dictionaries
- Real-world applications

## What is a Dictionary?

**Define: Dictionary** - An unordered collection of key-value pairs.

**Define: Key** - A unique identifier used to access a value.

**Define: Value** - The data associated with a key.

**Define: Key-Value Pair** - A key and its corresponding value, like `"name": "Alice"`.

**Define: Mutable** - Dictionaries can be changed after creation.

### Real-World Analogy

Think of a **real dictionary**:
- **Key:** The word you look up ("apple")
- **Value:** The definition ("a round fruit...")
- You find information by the word, not by page number!

Or think of a **phone book**:
- **Key:** Person's name
- **Value:** Phone number
- You search by name, not by position in the book

### Key Characteristics

**Dictionary Properties:**
1. **Unordered** - Items don't have a position (before Python 3.7; now insertion order is preserved)
2. **Mutable** - Can change, add, remove items
3. **Indexed by Keys** - Access values using keys, not numeric indices
4. **Keys Must Be Unique** - No duplicate keys allowed
5. **Keys Must Be Immutable** - Strings, numbers, tuples (not lists!)

```python
# Valid dictionary
person = {
    "name": "Alice",        # string key
    "age": 25,              # value is number
    "is_student": True,     # value is boolean
    "grades": [90, 85, 88]  # value can be a list!
}

# Keys are unique - last value wins
test = {"name": "Alice", "name": "Bob"}
print(test)  # {'name': 'Bob'}  ‚Üê 'Alice' was overwritten
```

## Creating Dictionaries

### Empty Dictionary

```python
# Method 1: Using curly braces
my_dict = {}

# Method 2: Using dict() function
my_dict = dict()

print(my_dict)  # {}
print(type(my_dict))  # <class 'dict'>
```

### Dictionary with Initial Values

```python
# Basic dictionary
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

print(person)
# {'name': 'Alice', 'age': 25, 'city': 'New York'}
```

### Different Value Types

```python
student = {
    "name": "Bob",           #  string
    "age": 20,               # number
    "grades": [85, 90, 88],  # list
    "is_active": True,       # boolean
    "address": {             # nested dictionary!
        "street": "123 Main St",
        "city": "Boston"
    }
}

print(student["name"])    # Bob
print(student["grades"])  # [85, 90, 88]
```

### Creating from Lists

```python
# Using dict() with list of tuples
pairs = [("name", "Alice"), ("age", 25), ("city", "NYC")]
person = dict(pairs)
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'NYC'}

# Using dict() with keyword arguments
person = dict(name="Alice", age=25, city="NYC")
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'NYC'}
```

### Using fromkeys()

```python
# Create dictionary with same value for multiple keys
keys = ["a", "b", "c"]
default_value = 0

my_dict = dict.fromkeys(keys, default_value)
print(my_dict)  # {'a': 0, 'b': 0, 'c': 0}
```

## Accessing Dictionary Values

### Using Square Brackets

```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Access values by key
print(person["name"])  # Alice
print(person["age"])   # 25
print(person["city"])  # New York
```

**‚ö†Ô∏è Important:** If key doesn't exist, you get an error!

```python
person = {"name": "Alice"}
# print(person["age"])  # KeyError: 'age'
```

### Using get() Method (Safer!)

**Define: get()** - Returns value for key, or a default if key doesn't exist.

```python
person = {"name": "Alice", "age": 25}

# Key exists
print(person.get("name"))  # Alice

# Key doesn't exist - returns None
print(person.get("phone"))  # None

# Key doesn't exist - returns custom default
print(person.get("phone", "Not found"))  # Not found
```

**Why get() is better:**
- No error if key missing
- Can provide default value
- Safer for dynamic data

### Checking if Key Exists

```python
person = {"name": "Alice", "age": 25}

# Check before accessing
if "name" in person:
    print("Name:", person["name"])  # Name: Alice

if "phone" in person:
    print("Phone:", person["phone"])
else:
    print("Phone not available")  # This prints

# Check if key NOT in dictionary
if "email" not in person:
    print("Email not found")  # This prints
```

## Modifying Dictionaries

**Dictionaries are mutable** - you can change them!

### Adding New Key-Value Pairs

```python
person = {"name": "Alice", "age": 25}
print("Before:", person)

# Add new key
person["city"] = "New York"
person["job"] = "Engineer"

print("After:", person)
# {'name': 'Alice', 'age': 25, 'city': 'New York', 'job': 'Engineer'}
```

### Updating Existing Values

```python
person = {"name": "Alice", "age": 25}
print("Before:", person)

# Update age
person["age"] = 26

print("After:", person)  # {'name': 'Alice', 'age': 26}
```

### Removing Items - del

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

del person["city"]
print(person)  # {'name': 'Alice', 'age': 25}

# del person["phone"]  # Error if key doesn't exist!
```

### Removing Items - pop()

**Define: pop()** - Removes key and returns its value.

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

# Remove and get the value
removed_value = person.pop("city")
print("Removed:", removed_value)  # NYC
print("Dict now:", person)  # {'name': 'Alice', 'age': 25}

# With default if key doesn't exist
result = person.pop("phone", "Not found")
print(result)  # Not found
```

### Removing Items - popitem()

**Define: popitem()** - Removes and returns the last inserted key-value pair.

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

item = person.popitem()
print("Removed:", item)  # ('city', 'NYC')
print("Dict now:", person)  # {'name': 'Alice', 'age': 25}
```

### Clearing All Items - clear()

```python
person = {"name": "Alice", "age": 25}

person.clear()
print(person)  # {}  ‚Üê empty dictionary
```

## Dictionary Methods

### keys() - Get All Keys

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

keys = person.keys()
print(keys)  # dict_keys(['name', 'age', 'city'])

# Convert to list
keys_list = list(person.keys())
print(keys_list)  # ['name', 'age', 'city']
```

### values() - Get All Values

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

values = person.values()
print(values)  # dict_values(['Alice', 25, 'NYC'])

# Convert to list
values_list = list(person.values())
print(values_list)  # ['Alice', 25, 'NYC']
```

### items() - Get All Key-Value Pairs

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

items = person.items()
print(items)
# dict_items([('name', 'Alice'), ('age', 25), ('city', 'NYC')])

# Convert to list of tuples
items_list = list(person.items())
print(items_list)
# [('name', 'Alice'), ('age', 25), ('city', 'NYC')]
```

### update() - Merge Dictionaries

**Define: update()** - Adds key-value pairs from another dictionary.

```python
person = {"name": "Alice", "age": 25}
additional = {"city": "NYC", "job": "Engineer"}

person.update(additional)
print(person)
# {'name': 'Alice', 'age': 25, 'city': 'NYC', 'job': 'Engineer'}

# Updates existing keys too
person.update({"age": 26})
print(person["age"])  # 26
```

### copy() - Create a Copy

```python
original = {"name": "Alice", "age": 25}

# Create a copy
copy_dict = original.copy()

# Modify copy
copy_dict["age"] = 30

print("Original:", original)  # {'name': 'Alice', 'age': 25}
print("Copy:", copy_dict)     # {'name': 'Alice', 'age': 30}
```

### setdefault() - Get or Set Default

**Define: setdefault()** - Returns value if key exists; otherwise, sets and returns default.

```python
person = {"name": "Alice"}

# Key exists - returns existing value
name = person.setdefault("name", "Unknown")
print(name)  # Alice

# Key doesn't exist - adds it with default value
age = person.setdefault("age", 0)
print(age)  # 0
print(person)  # {'name': 'Alice', 'age': 0}
```

## Looping Through Dictionaries

### Loop Through Keys (Default)

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

# Default - iterates through keys
for key in person:
    print(key)

# Output:
# name
# age
# city
```

### Loop Through Keys Explicitly

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

for key in person.keys():
    print(key, ":", person[key])

# Output:
# name : Alice
# age : 25
# city : NYC
```

### Loop Through Values

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

for value in person.values():
    print(value)

# Output:
# Alice
# 25
# NYC
```

### Loop Through Key-Value Pairs (Best!)

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}

for key, value in person.items():
    print(key, ":", value)

# Output:
# name : Alice
# age : 25
# city : NYC
```

### Building Dictionaries with Loops

```python
# Create squares dictionary
squares = {}
for i in range(1, 6):
    squares[i] = i ** 2

print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

### Dictionary Comprehension

**Define: Dictionary comprehension** - Create dictionaries in one line.

```python
# Traditional way
squares = {}
for i in range(1, 6):
    squares[i] = i ** 2

# Dictionary comprehension
squares = {i: i**2 for i in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
```

**More examples:**

```python
# Even numbers and their squares
evens = {i: i**2 for i in range(11) if i % 2 == 0}
print(evens)  # {0: 0, 2: 4, 4: 16, 6: 36, 8: 64, 10: 100}

# Uppercase fruits
fruits = ["apple", "banana", "cherry"]
fruit_upper = {fruit: fruit.upper() for fruit in fruits}
print(fruit_upper)
# {'apple': 'APPLE', 'banana': 'BANANA', 'cherry': 'CHERRY'}
```

## Nested Dictionaries

**Define: Nested dictionary** - A dictionary containing other dictionaries as values.

### Creating Nested Dictionaries

```python
# Students database
students = {
    "student1": {
        "name": "Alice",
        "age": 20,
        "grades": [85, 90, 88]
    },
    "student2": {
        "name": "Bob",
        "age": 22,
        "grades": [78, 85, 82]
    },
    "student3": {
        "name": "Charlie",
        "age": 21,
        "grades": [92, 95, 89]
    }
}

print(students)
```

### Accessing Nested Values

```python
students = {
    "student1": {
        "name": "Alice",
        "age": 20,
        "grades": [85, 90, 88]
    }
}

# Access nested values
print(students["student1"])  # {'name': 'Alice', 'age': 20, 'grades': [85, 90, 88]}
print(students["student1"]["name"])  # Alice
print(students["student1"]["grades"])  # [85, 90, 88]
print(students["student1"]["grades"][0])  # 85
```

### Modifying Nested Values

```python
students = {
    "student1": {
        "name": "Alice",
        "age": 20
    }
}

# Modify nested value
students["student1"]["age"] = 21
print(students["student1"]["age"])  # 21

# Add nested value
students["student1"]["city"] = "NYC"
print(students["student1"])
# {'name': 'Alice', 'age': 21, 'city': 'NYC'}
```

### Looping Through Nested Dictionaries

```python
students = {
    "student1": {"name": "Alice", "age": 20},
    "student2": {"name": "Bob", "age": 22}
}

for student_id, student_info in students.items():
    print("Student ID:", student_id)
    for key, value in student_info.items():
        print("  ", key, ":", value)
    print()

# Output:
# Student ID: student1
#    name : Alice
#    age : 20
#
# Student ID: student2
#    name : Bob
#    age : 22
```

## Dictionary Operations

### Length

```python
person = {"name": "Alice", "age": 25, "city": "NYC"}
print(len(person))  # 3
```

### Membership Testing

```python
person = {"name": "Alice", "age": 25}

print("name" in person)     # True  ‚Üê checks keys
print("Alice" in person)    # False ‚Üê 'Alice' is a value, not a key
print("phone" not in person)  # True
```

### Merging Dictionaries

```python
# Method 1: update()
dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}
dict1.update(dict2)
print(dict1)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# Method 2: ** unpacking (Python 3.5+)
dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}
merged = {**dict1, **dict2}
print(merged)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}
```

### Comparing Dictionaries

```python
dict1 = {"name": "Alice", "age": 25}
dict2 = {"name": "Alice", "age": 25}
dict3 = {"name": "Bob", "age": 30}

print(dict1 == dict2)  # True  ‚Üê same keys and values
print(dict1 == dict3)  # False ‚Üê different values
```

### Example 3: Word Counter

```python
def count_words(text):
    word_count = {}
    words = text.split()
    
    for word in words:
        # Convert to lowercase for consistency
        word = word.lower()
        
        # Count occurrences
        if word in word_count:
            word_count[word] = word_count[word] + 1
        else:
            word_count[word] = 1
    
    return word_count

text = "hello world hello python world world"
counts = count_words(text)

for word, count in counts.items():
    print(word, ":", count)

# Output:
# hello : 2
# world : 3
# python : 1
```

## Common Mistakes

### Mistake 1: Using Mutable Keys

**‚ùå Wrong Way:**
```python
# Lists cannot be keys!
# my_dict = {[1, 2]: "value"}  # Error! TypeError: unhashable type: 'list'
```

**‚úÖ Correct Way:**
```python
# Use tuples instead
my_dict = {(1, 2): "value"}
print(my_dict)  # {(1, 2): 'value'}
```

### Mistake 2: Accessing Non-Existent Keys

**‚ùå Wrong Way:**
```python
person = {"name": "Alice"}
# print(person["age"])  # KeyError: 'age'
```

**‚úÖ Correct Way:**
```python
person = {"name": "Alice"}

# Method 1: Check first
if "age" in person:
    print(person["age"])
else:
    print("Age not found")

# Method 2: Use get()
print(person.get("age", "Not found"))
```

### Mistake 3: Forgetting Dictionaries Are Mutable

**‚ùå Wrong Way:**
```python
dict1 = {"a": 1, "b": 2}
dict2 = dict1  # NOT a copy!

dict2["c"] = 3
print(dict1)  # {'a': 1, 'b': 2, 'c': 3}  ‚Üê Oops!
```

**‚úÖ Correct Way:**
```python
dict1 = {"a": 1, "b": 2}
dict2 = dict1.copy()  # Make a copy

dict2["c"] = 3
print(dict1)  # {'a': 1, 'b': 2}  ‚Üê Unchanged
print(dict2)  # {'a': 1, 'b': 2, 'c': 3}
```

### Mistake 4: Modifying Dictionary During Iteration

**‚ùå Wrong Way:**
```python
my_dict = {"a": 1, "b": 2, "c": 3}
for key in my_dict:
    if my_dict[key] % 2 == 0:
        del my_dict[key]  # Error! Dictionary changed size during iteration
```

**‚úÖ Correct Way:**
```python
my_dict = {"a": 1, "b": 2, "c": 3}
to_delete = []

for key in my_dict:
    if my_dict[key] % 2 == 0:
        to_delete.append(key)

for key in to_delete:
    del my_dict[key]

print(my_dict)  # {'a': 1, 'c': 3}
```

### Mistake 5: Confusing Keys and Values

**‚ùå Wrong Way:**
```python
person = {"name": "Alice", "age": 25}
print("Alice" in person)  # False  ‚Üê "Alice" is a value, not a key!
```

**‚úÖ Correct Way:**
```python
person = {"name": "Alice", "age": 25}

# Check keys
print("name" in person)  # True

# Check values
print("Alice" in person.values())  # True
```

## Best Practices

### ‚úÖ Do This:

1. **Use Descriptive Keys**
   ```python
   # Good
   person = {"first_name": "Alice", "last_name": "Johnson"}
   
   # Avoid
   person = {"fn": "Alice", "ln": "Johnson"}
   ```

2. **Use get() for Safe Access**
   ```python
   # Good
   value = my_dict.get("key", default_value)
   
   # Risky
   value = my_dict["key"]  # Error if key missing
   ```

3. **Use items() When Looping**
   ```python
   # Good - clear and Pythonic
   for key, value in my_dict.items():
       print(key, value)
   
   # Less clear
   for key in my_dict:
       print(key, my_dict[key])
   ```

4. **Use Dictionary Comprehension When Simple**
   ```python
   # Good for simple operations
   squares = {x: x**2 for x in range(5)}
   ```

5. **Document Nested Structures**
   ```python
   # Good - clear structure
   student = {
       "name": "Alice",
       "grades": {
           "math": 90,
           "english": 85
       }
   }
   ```

### ‚ùå Avoid This:

1. **Don't Use Lists or Dicts as Keys**
   ```python
   # Avoid
   # my_dict = {[1, 2]: "value"}  # Error!
   
   # Use tuples instead
   my_dict = {(1, 2): "value"}
   ```

2. **Don't Modify During Iteration**
   ```python
   # Avoid
   for key in my_dict:
       del my_dict[key]  # Dangerous!
   ```

3. **Don't Forget to Check Key Existence**
   ```python
   # Avoid
   value = my_dict["might_not_exist"]
   
   # Better
   value = my_dict.get("might_not_exist", default)
   ```

## Summary

Congratulations! You've mastered Python dictionaries - the most versatile data structure! üéâ

**üéØ Key Takeaways:**

**üìå Dictionary Basics**
- Unordered collection of key-value pairs
- Created with curly braces: `{key: value}`
- Mutable (can change)
- Keys must be unique and immutable
- Values can be any type

**üìå Creating Dictionaries**
- Empty: `{}`or `dict()`
- With values: `{"key": "value"}`
- From tuples: `dict([(k, v)])`
- Comprehension: `{k: v for k, v in...}`

**üìå Accessing Values**
- Direct: `dict["key"]` (error if missing)
- Safe: `dict.get("key", default)` (no error)
- Check: `"key" in dict`

**üìå Modifying**
- Add/Update: `dict["key"] = value`
- Remove: `del dict["key"]`
- Pop: `dict.pop("key")`
- Clear: `dict.clear()`

**üìå Important Methods**
- `keys()` - all keys
- `values()` - all values
- `items()` - all key-value pairs
- `update()` - merge dictionaries
- `get()` - safe access
- `pop()` - remove and return
- `copy()` - create copy

**üìå Looping**
- Keys: `for key in dict:`
- Values: `for value in dict.values():`
- Pairs: `for key, value in dict.items():`

**üìå Nested Dictionaries**
- Dictionaries within dictionaries
- Access: `dict["outer"]["inner"]`
- Great for complex data structures

**üí° Remember:**
- Keys must be immutable (strings, numbers, tuples)
- Keys must be unique
- Use get() for safe access
- Perfect for lookups by name
- Unordered (but insertion order preserved in Python 3.7+)
- Mutable - can change anytime

**üîë Quick Reference:**

| Operation | Syntax | Example |
|-----------|--------|---------|
| Create | `{}` or `dict()` | `{"a": 1}` |
| Access | `dict[key]` | `person["name"]` |
| Safe access | `dict.get(key)` | `person.get("age", 0)` |
| Add/Update | `dict[key] = value` | `person["city"] = "NYC"` |
| Remove | `del dict[key]` | `del person["age"]` |
| Check key | `key in dict` | `"name" in person` |
| Loop items | `for k, v in dict.items():` | See examples |
