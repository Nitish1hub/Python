## What is a Condition?

Before we learn about `if` statements, we need to understand **conditions**.

**Define: Condition** - A statement that can be either **True** or **False**.

Think of it like asking a yes/no question:
- "Is it raining?" â†’ Yes (True) or No (False)
- "Are you hungry?" â†’ Yes (True) or No (False)
- "Is the number greater than 10?" â†’ Yes (True) or No (False)

In Python, conditions use **comparison operators** (which we learned in Chapter 2):

### Quick Review of Comparison Operators

| Operator | Meaning | Example | Result |
|----------|---------|---------|--------|
| `==` | Equal to | `5 == 5` | True |
| `!=` | Not equal to | `5 != 3` | True |
| `>` | Greater than | `5 > 3` | True |
| `<` | Less than | `5 < 3` | False |
| `>=` | Greater than or equal | `5 >= 5` | True |
| `<=` | Less than or equal | `3 <= 5` | True |

### Condition Examples

```python
# These are all conditions (they evaluate to True or False)

age = 18
print(age >= 18)  # True

temperature = 25
print(temperature > 30)  # False

name = "Alice"
print(name == "Alice")  # True

score = 85
print(score < 50)  # False
```

**Key Point:** A condition always gives us either `True` or `False`.

## The `if` Statement

Now that we understand conditions, let's use them to make decisions!

**Define: `if` statement** - A way to run code ONLY if a condition is True.

### Real-World Analogy

Think of an automatic door:
- **IF** someone approaches â†’ the door opens
- **IF** nobody is there â†’ the door stays closed

### Basic Syntax

```python
if condition:
    # This code runs ONLY if condition is True
    # Notice the indentation (4 spaces or 1 tab)
```

**Important parts:**
1. `if` keyword
2. Condition (something that is True or False)
3. Colon (`:`) at the end
4. **Indentation** - the code inside must be indented

**Define: Indentation** - Spaces at the beginning of a line that show which code belongs to the `if` statement.

### Simple Example

```python
age = 20

if age >= 18:
    print("You are an adult")
    print("You can vote")
    
# Output:
# You are an adult
# You can vote
```

**What happened?**
1. Python checks: Is `age >= 18`?
2. Yes! (20 is greater than or equal to 18)
3. So Python runs the indented code
4. Both print statements execute

### Example Where Condition is False

```python
age = 15

if age >= 18:
    print("You are an adult")
    print("You can vote")
    
print("This always runs")

# Output:
# This always runs
```

**What happened?**
1. Python checks: Is `age >= 18`?
2. No! (15 is less than 18)
3. So Python SKIPS the indented code
4. Only the last print runs (it's not indented, so it always runs)

### More Examples

```python
# Example 1: Temperature check
temperature = 35

if temperature > 30:
    print("It's hot outside!")
    print("Drink water and stay cool")

# Output:
# It's hot outside!
# Drink water and stay cool
```

```python
# Example 2: Password check
password = "secret123"
user_input = "secret123"

if password == user_input:
    print("Access granted!")
    print("Welcome back!")

# Output:
# Access granted!
# Welcome back!
```

```python
# Example 3: Number check
number = 42

if number % 2 == 0:
    print("This is an even number")

# Output:
# This is an even number
```

## The `else` Statement

What if we want to do something DIFFERENT when the condition is False?

**Define: `else` statement** - Code that runs when the `if` condition is False.

### Real-World Analogy

Think of choosing ice cream flavors:
- **IF** they have chocolate â†’ get chocolate
- **ELSE** (if they don't have chocolate) â†’ get vanilla

### Syntax

```python
if condition:
    # Runs if condition is True
else:
    # Runs if condition is False
```

### Example

```python
age = 15

if age >= 18:
    print("You are an adult")
    print("You can vote")
else:
    print("You are a minor")
    print("You cannot vote yet")

# Output:
# You are a minor
# You cannot vote yet
```

**What happened?**
1. Python checks: Is `age >= 18`?
2. No! (15 is less than 18)
3. So Python SKIPS the `if` block
4. Python runs the `else` block instead

### More if-else Examples

```python
# Example 1: Pass or Fail
score = 45

if score >= 50:
    print("You passed!")
else:
    print("You failed. Please try again.")

# Output:
# You failed. Please try again.
```

```python
# Example 2: Positive or Negative
number = -5

if number >= 0:
    print("This is a positive number")
else:
    print("This is a negative number")

# Output:
# This is a negative number
```

```python
# Example 3: Login System
correct_password = "python123"
user_password = "wrong_password"

if user_password == correct_password:
    print("Login successful!")
    print("Welcome to your account")
else:
    print("Incorrect password")
    print("Please try again")

# Output:
# Incorrect password
# Please try again
```

## The `elif` Statement

What if we have MORE than two choices?

**Define: `elif`** - Short for "else if". It checks another condition if the previous `if` was False.

### Real-World Analogy

Think of a restaurant menu:
- **IF** you want breakfast â†’ serve breakfast menu
- **ELIF** (else if) you want lunch â†’ serve lunch menu
- **ELIF** (else if) you want dinner â†’ serve dinner menu
- **ELSE** â†’ sorry, we're closed

### Syntax

```python
if condition1:
    # Runs if condition1 is True
elif condition2:
    # Runs if condition1 is False AND condition2 is True
elif condition3:
    # Runs if condition1 and condition2 are False AND condition3 is True
else:
    # Runs if all conditions are False
```

### Example: Grade Calculator

```python
score = 75

if score >= 90:
    print("Grade: A")
    print("Excellent work!")
elif score >= 80:
    print("Grade: B")
    print("Very good!")
elif score >= 70:
    print("Grade: C")
    print("Good job!")
elif score >= 60:
    print("Grade: D")
    print("You passed")
else:
    print("Grade: F")
    print("Failed. Please study more")

# Output:
# Grade: C
# Good job!
```

**How it works:**
1. Check: Is `score >= 90`? â†’ No (75 is less than 90)
2. Check: Is `score >= 80`? â†’ No (75 is less than 80)
3. Check: Is `score >= 70`? â†’ Yes! (75 is greater than or equal to 70)
4. Run that block and STOP checking the rest

**Important:** Once one condition is True, Python stops checking the others!

### More elif Examples

## Nested Conditions

Sometimes we need to check a condition INSIDE another condition.

**Define: Nested conditions** - An `if` statement inside another `if` statement.

### Real-World Analogy

Think of entering a building:
- **IF** you have an access card:
  - **IF** the card is valid â†’ enter
  - **ELSE** â†’ card expired, access denied
- **ELSE** â†’ no card, access denied

### Syntax

```python
if condition1:
    # This runs if condition1 is True
    
    if condition2:
        # This runs if BOTH condition1 AND condition2 are True
    else:
        # This runs if condition1 is True but condition2 is False
else:
    # This runs if condition1 is False
```

### Example: Movie Theater

```python
age = 16
has_ticket = True

if has_ticket:
    print("You have a ticket")
    
    if age >= 18:
        print("You can watch the R-rated movie")
    else:
        print("You can only watch PG movies")
else:
    print("Please buy a ticket first")

# Output:
# You have a ticket
# You can only watch PG movies
```

### More Nested Examples

```python
# Example 1: Login with username and password
username = "admin"
password = "pass123"

user_input_name = "admin"
user_input_pass = "pass123"

if user_input_name == username:
    print("Username correct")
    
    if user_input_pass == password:
        print("Password correct")
        print("Login successful!")
    else:
        print("Wrong password")
else:
    print("Username not found")

# Output:
# Username correct
# Password correct
# Login successful!
```

```python
# Example 2: Discount Calculator
is_member = True
purchase_amount = 150

if purchase_amount >= 100:
    print("Eligible for discount!")
    
    if is_member:
        print("Member discount: 20%")
        discount = purchase_amount * 0.20
    else:
        print("Non-member discount: 10%")
        discount = purchase_amount * 0.10
    
    final_price = purchase_amount - discount
    print("Final price:", final_price)
else:
    print("No discount available")
    print("Minimum purchase: $100")

# Output:
# Eligible for discount!
# Member discount: 20%
# Final price: 120.0
```

**Note:** Try to avoid too many nested levels. It makes code hard to read!

## Combining Conditions with Logical Operators

We can combine multiple conditions using **logical operators** (from Chapter 2).

### Quick Review of Logical Operators

| Operator | Meaning | Example |
|----------|---------|---------|
| `and` | Both conditions must be True | `age >= 18 and has_license` |
| `or` | At least one must be True | `is_weekend or is_holiday` |
| `not` | Reverses the condition | `not is_raining` |

### Using `and` Operator

```python
# Both conditions must be True

age = 20
has_license = True

if age >= 18 and has_license:
    print("You can drive!")
else:
    print("You cannot drive")

# Output:
# You can drive!
```

### Using `or` Operator

```python
# At least one condition must be True

day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")
    print("Time to relax")
else:
    print("It's a weekday")
    print("Time to work")

# Output:
# It's the weekend!
# Time to relax
```

### Using `not` Operator

```python
# Reverses the condition

is_raining = False

if not is_raining:
    print("No rain today!")
    print("Perfect for a picnic")
else:
    print("It's raining")
    print("Stay inside")

# Output:
# No rain today!
# Perfect for a picnic
```

### Complex Combinations

```python
# Example: Scholarship Eligibility
score = 85
attendance = 90
behavior = "good"

if score >= 80 and attendance >= 85 and behavior == "good":
    print("Congratulations! You qualify for the scholarship!")
else:
    print("Sorry, you don't meet the requirements")

# Output:
# Congratulations! You qualify for the scholarship!
```

```python
# Example: Weekend or Holiday Check

is_holiday = False

if day == "Saturday" or day == "Sunday" or is_holiday:
    print("Enjoy your day off!")
else:
    print("Time for work")

# Output:
# Time for work
```

## Common Mistakes

Let's look at errors beginners often make!

### Mistake 1: Forgetting the Colon

**âŒ Wrong Way:**
```python
age = 18
if age >= 18  # Missing colon!
    print("Adult")
```

**Error Message:**
```
SyntaxError: invalid syntax
```

**âœ… Correct Way:**
```python
age = 18
if age >= 18:  # Don't forget the colon!
    print("Adult")
```

### Mistake 2: Wrong Indentation

**âŒ Wrong Way:**
```python
age = 18
if age >= 18:
print("Adult")  # Not indented!
```

**Error Message:**
```
IndentationError: expected an indented block
```

**âœ… Correct Way:**
```python
age = 18
if age >= 18:
    print("Adult")  # Properly indented (4 spaces)
```

### Mistake 3: Using = Instead of ==

**âŒ Wrong Way:**
```python
age = 18
if age = 18:  # Single = is assignment, not comparison!
    print("Adult")
```

**Error Message:**
```
SyntaxError: invalid syntax
```

**âœ… Correct Way:**
```python
age = 18
if age == 18:  # Double == for comparison
    print("Adult")
```

### Mistake 4: Confusing elif and else Order

**âŒ Wrong Way:**
```python
score = 75
if score >= 90:
    print("A")
else:
    print("Not A")
elif score >= 80:  # elif cannot come after else!
    print("B")
```

**âœ… Correct Way:**
```python
score = 75
if score >= 90:
    print("A")
elif score >= 80:  # elif comes BEFORE else
    print("B")
else:
    print("Not A or B")
```

### Mistake 5: Missing Indentation in else

**âŒ Wrong Way:**
```python
age = 15
if age >= 18:
    print("Adult")
else:
print("Minor")  # else block not indented!
```

**âœ… Correct Way:**
```python
age = 15
if age >= 18:
    print("Adult")
else:
    print("Minor")  # Properly indented
```

## Best Practices

### âœ… Do This:

1. **Use Meaningful Condition Names**
   ```python
   # Good
   is_eligible = age >= 18 and is_citizen
   if is_eligible:
       print("Can vote")
   
   # Instead of
   if age >= 18 and is_citizen:
       print("Can vote")
   ```

2. **Keep Conditions Simple**
   ```python
   # Good - easier to read
   is_adult = age >= 18
   has_permission = is_citizen and has_id
   
   if is_adult and has_permission:
       print("Access granted")
   
   # Instead of complex one-liner
   if age >= 18 and is_citizen and has_id:
       print("Access granted")
   ```

3. **Use Consistent Indentation**
   - Always use 4 spaces (or 1 tab)
   - Be consistent throughout your code
   - Most code editors do this automatically

4. **Add Comments for Complex Logic**
   ```python
   # Check if user qualifies for premium membership
   if total_purchases >= 1000 and account_age > 365:
       membership_type = "Premium"
   ```

5. **Handle All Cases**
   ```python
   # Good - handles all possibilities
   if score >= 50:
       print("Pass")
   else:
       print("Fail")
   
   # Instead of leaving cases unhandled
   if score >= 50:
       print("Pass")
   # What if score < 50? No message!
   ```

### âŒ Avoid This:

1. **Too Many Nested Levels**
   ```python
   # Avoid - hard to read
   if condition1:
       if condition2:
           if condition3:
               if condition4:
                   # Too deep!
   ```

2. **Comparing Boolean Values**
   ```python
   # Avoid
   if is_active == True:
       print("Active")
   
   # Better
   if is_active:
       print("Active")
   ```

3. **Redundant Conditions**
   ```python
   # Avoid
   if age >= 18:
       is_adult = True
   else:
       is_adult = False
   
   # Better
   is_adult = age >= 18
   ```

## Summary

**ðŸ“Œ if Statement**
- Runs code ONLY if condition is True
