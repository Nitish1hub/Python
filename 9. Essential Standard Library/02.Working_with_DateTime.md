## The datetime Module

Python's `datetime` module provides classes for working with dates and times.

### Main Classes

| Class | Purpose | Example |
|-------|---------|---------|
| `datetime` | Date and time combined | 2024-01-15 14:30:00 |
| `date` | Just the date | 2024-01-15 |
| `time` | Just the time | 14:30:00 |
| `timedelta` | Duration/difference between dates | 5 days, 3 hours |
| `timezone` | Timezone information | UTC, EST |

### Importing

```python
from datetime import datetime, date, time, timedelta
```

## Formatting Dates and Times

**Define: strftime** - "String format time" - converts datetime to string.

### Common Format Codes

| Code | Meaning | Example |
|------|---------|---------|
| `%Y` | Year (4 digits) | 2024 |
| `%y` | Year (2 digits) | 24 |
| `%m` | Month (01-12) | 01 |
| `%B` | Month name | January |
| `%b` | Month abbrev | Jan |
| `%d` | Day (01-31) | 15 |
| `%A` | Weekday name | Monday |
| `%a` | Weekday abbrev | Mon |
| `%H` | Hour (00-23) | 14 |
| `%I` | Hour (01-12) | 02 |
| `%p` | AM/PM | PM |
| `%M` | Minute | 30 |
| `%S` | Second | 45 |

### Formatting Examples

```python
from datetime import datetime

now = datetime.now()

# Different formats
print(now.strftime("%Y-%m-%d"))           # 2024-01-15
print(now.strftime("%d/%m/%Y"))           # 15/01/2024
print(now.strftime("%B %d, %Y"))          # January 15, 2024
print(now.strftime("%A, %B %d, %Y"))      # Monday, January 15, 2024
print(now.strftime("%I:%M %p"))           # 02:30 PM
print(now.strftime("%H:%M:%S"))           # 14:30:45

# Custom format
print(now.strftime("%Y-%m-%d %H:%M:%S"))  # 2024-01-15 14:30:45
print(now.strftime("%a, %b %d at %I:%M %p"))  # Mon, Jan 15 at 02:30 PM
```

### Common Date Formats

```python
from datetime import datetime

now = datetime.now()

# US format
print(now.strftime("%m/%d/%Y"))  # 01/15/2024

# UK format
print(now.strftime("%d/%m/%Y"))  # 15/01/2024

# Full text
print(now.strftime("%A, %B %d, %Y at %I:%M %p"))
# Monday, January 15, 2024 at 02:30 PM
```

## Parsing Date Strings

**Define: strptime** - "String parse time" - converts string to datetime.

### Basic Parsing

```python
from datetime import datetime

# Parse date string
date_string = "2024-01-15"
date_obj = datetime.strptime(date_string, "%Y-%m-%d")
print(date_obj)  # 2024-01-15 00:00:00
print(type(date_obj))  # <class 'datetime.datetime'>

# Parse datetime string
datetime_string = "2024-01-15 14:30:45"
datetime_obj = datetime.strptime(datetime_string, "%Y-%m-%d %H:%M:%S")
print(datetime_obj)  # 2024-01-15 14:30:45
```

### Parsing Different Formats

```python
from datetime import datetime

# US format
us_date = "01/15/2024"
parsed = datetime.strptime(us_date, "%m/%d/%Y")
print(parsed)  # 2024-01-15 00:00:00

# UK format
uk_date = "15/01/2024"
parsed = datetime.strptime(uk_date, "%d/%m/%Y")
print(parsed)  # 2024-01-15 00:00:00

# Text format
text_date = "January 15, 2024"
parsed = datetime.strptime(text_date, "%B %d, %Y")
print(parsed)  # 2024-01-15 00:00:00

# With time
full_string = "Mon, Jan 15 at 02:30 PM"
parsed = datetime.strptime(full_string, "%a, %b %d at %I:%M %p")
print(parsed)  # 1900-01-15 14:30:00 (note: year defaults to 1900)
```

### Error Handling

```python
from datetime import datetime

def parse_date_safe(date_string, format_string):
    """Safely parse date string"""
    try:
        return datetime.strptime(date_string, format_string)
    except ValueError as e:
        print(f"Error parsing date: {e}")
        return None

# Valid date
result = parse_date_safe("2024-01-15", "%Y-%m-%d")
print(result)  # 2024-01-15 00:00:00

# Invalid date
result = parse_date_safe("2024-13-45", "%Y-%m-%d")
# Error parsing date: day is out of range for month
```

## Time Deltas - Date Arithmetic

**Define: timedelta** - Represents a duration or difference between dates/times.

### Creating Time Deltas

```python
from datetime import timedelta

# Create time deltas
one_day = timedelta(days=1)
one_week = timedelta(weeks=1)
one_hour = timedelta(hours=1)
thirty_minutes = timedelta(minutes=30)

# Combined
duration = timedelta(days=7, hours=3, minutes=30, seconds=45)
print(duration)  # 7 days, 3:30:45
```

### Date Arithmetic

```python
from datetime import datetime, timedelta

now = datetime.now()
print(f"Now: {now}")

# Add time
tomorrow = now + timedelta(days=1)
print(f"Tomorrow: {tomorrow}")

next_week = now + timedelta(weeks=1)
print(f"Next week: {next_week}")

# Subtract time
yesterday = now - timedelta(days=1)
print(f"Yesterday: {yesterday}")

one_hour_ago = now - timedelta(hours=1)
print(f"One hour ago: {one_hour_ago}")
```

### Calculating Differences

```python
from datetime import datetime, date

# Date difference
today = date.today()
birthday = date(1990, 5, 15)

age_delta = today - birthday
print(f"Days old: {age_delta.days}")
print(f"Years old: {age_delta.days // 365}")

# DateTime difference
now = datetime.now()
past = datetime(2024, 1, 1, 0, 0, 0)

difference = now - past
print(f"Days: {difference.days}")
print(f"Total seconds: {difference.total_seconds()}")
```

### Accessing TimeDelta Components

```python
from datetime import timedelta

duration = timedelta(days=7, hours=3, minutes=30, seconds=45)

print(f"Days: {duration.days}")  # 7
print(f"Seconds: {duration.seconds}")  # 12645 (hours+minutes+seconds in seconds)
print(f"Total seconds: {duration.total_seconds()}")  # 617445.0

# Calculate hours from seconds
hours = duration.seconds // 3600
minutes = (duration.seconds % 3600) // 60
seconds = duration.seconds % 60

print(f"Breakdown: {duration.days} days, {hours} hours, {minutes} minutes, {seconds} seconds")
```

## Comparing Dates and Times

### Comparison Operators

```python
from datetime import datetime, date

# Comparing dates
date1 = date(2024, 1, 15)
date2 = date(2024, 12, 25)

print(date1 < date2)   # True
print(date1 > date2)   # False
print(date1 == date2)  # False
print(date1 != date2)  # True

# Comparing datetimes
dt1 = datetime(2024, 1, 15, 10, 30)
dt2 = datetime(2024, 1, 15, 14, 30)

print(dt1 < dt2)   # True (earlier time)
print(dt1 == dt2)  # False
```

### Finding Min/Max Dates

```python
from datetime import date

dates = [
    date(2024, 1, 15),
    date(2024, 12, 25),
    date(2023, 6, 10),
    date(2025, 3, 20)
]

earliest = min(dates)
latest = max(dates)

print(f"Earliest: {earliest}")  # 2023-06-10
print(f"Latest: {latest}")      # 2025-03-20
```

### Checking if Date is in Range

```python
from datetime import date

today = date.today()
start_date = date(2024, 1, 1)
end_date = date(2024, 12, 31)

if start_date <= today <= end_date:
    print("Date is in 2024!")
```

## Working with Timezones

### Timezone-Aware vs Naive

```python
from datetime import datetime, timezone

# Naive datetime (no timezone)
naive = datetime.now()
print(naive)  # 2024-01-15 14:30:45.123456
print(naive.tzinfo)  # None

# Timezone-aware datetime
aware = datetime.now(timezone.utc)
print(aware)  # 2024-01-15 19:30:45.123456+00:00
print(aware.tzinfo)  # UTC
```

### UTC Timezone

```python
from datetime import datetime, timezone

# Current UTC time
utc_now = datetime.now(timezone.utc)
print(utc_now)  # 2024-01-15 19:30:45.123456+00:00

# Convert naive to UTC
naive = datetime.now()
utc_aware = naive.replace(tzinfo=timezone.utc)
print(utc_aware)
```

### Custom Timezones

```python
from datetime import datetime, timezone, timedelta

# Create timezone with offset
# EST is UTC-5
est = timezone(timedelta(hours=-5))
est_time = datetime.now(est)
print(est_time)  # 2024-01-15 14:30:45.123456-05:00

# PST is UTC-8
pst = timezone(timedelta(hours=-8))
pst_time = datetime.now(pst)
print(pst_time)  # 2024-01-15 11:30:45.123456-08:00
```

### Converting Between Timezones

```python
from datetime import datetime, timezone, timedelta

# UTC time
utc_time = datetime.now(timezone.utc)
print(f"UTC: {utc_time}")

# Convert to EST (UTC-5)
est = timezone(timedelta(hours=-5))
est_time = utc_time.astimezone(est)
print(f"EST: {est_time}")

# Convert to PST (UTC-8)
pst = timezone(timedelta(hours=-8))
pst_time = utc_time.astimezone(pst)
print(f"PST: {pst_time}")
```

## Real-Life Practical Examples

### Example 1: Age Calculator

```python
from datetime import date

def calculate_age(birth_date):
    """Calculate age from birth date"""
    today = date.today()
    
    age = today.year - birth_date.year
    
    # Adjust if birthday hasn't occurred this year
    if (today.month, today.day) < (birth_date.month, birth_date.day):
        age -= 1
    
    return age

birthday = date(1990, 5, 15)
age = calculate_age(birthday)
print(f"Age: {age} years old")
```

### Example 2: Time Until Event

```python
from datetime import datetime

def time_until_event(event_datetime):
    """Calculate time remaining until event"""
    now = datetime.now()
    
    if event_datetime < now:
        return "Event has passed!"
    
    delta = event_datetime - now
    
    days = delta.days
    hours, remainder = divmod(delta.seconds, 3600)
    minutes, seconds = divmod(remainder, 60)
    
    return f"{days} days, {hours} hours, {minutes} minutes, {seconds} seconds"

# New Year 2025
new_year = datetime(2025, 1, 1, 0, 0, 0)
print(f"Time until New Year: {time_until_event(new_year)}")
```

### Example 3: Business Days Calculator

```python
from datetime import date, timedelta

def add_business_days(start_date, days):
    """Add business days (excluding weekends)"""
    current = start_date
    days_added = 0
    
    while days_added < days:
        current += timedelta(days=1)
        # Skip weekends (5=Saturday, 6=Sunday)
        if current.weekday() < 5:
            days_added += 1
    
    return current

today = date.today()
deadline = add_business_days(today, 10)
print(f"10 business days from now: {deadline}")
```

### Example 5: Meeting Scheduler

```python
from datetime import datetime, timedelta

class Meeting:
    def __init__(self, title, start_time, duration_minutes):
        self.title = title
        self.start_time = start_time
        self.duration = timedelta(minutes=duration_minutes)
        self.end_time = start_time + self.duration
    
    def overlaps_with(self, other):
        """Check if this meeting overlaps with another"""
        return (self.start_time < other.end_time and 
                self.end_time > other.start_time)
    
    def __str__(self):
        return f"{self.title}: {self.start_time.strftime('%I:%M %p')} - {self.end_time.strftime('%I:%M %p')}"

# Create meetings
meeting1 = Meeting("Team Standup", datetime(2024, 1, 15, 9, 0), 30)
meeting2 = Meeting("Client Call", datetime(2024, 1, 15, 9, 20), 60)
meeting3 = Meeting("Lunch", datetime(2024, 1, 15, 12, 0), 60)

print(meeting1)
print(meeting2)
print(meeting3)

# Check overlap
if meeting1.overlaps_with(meeting2):
    print(f"\n‚ö†Ô∏è Conflict: {meeting1.title} overlaps with {meeting2.title}")
```

### Example 6: Date Range Generator

```python
from datetime import date, timedelta

def date_range(start_date, end_date):
    """Generate all dates between start and end"""
    current = start_date
    while current <= end_date:
        yield current
        current += timedelta(days=1)

# Get all dates in January 2024
start = date(2024, 1, 1)
end = date(2024, 1, 31)

print("All dates in January 2024:")
for d in date_range(start, end):
    print(d.strftime("%A, %B %d, %Y"))
```

## Common Mistakes

### Mistake 1: Forgetting Timezone Awareness

```python
from datetime import datetime, timezone

# ‚ùå Comparing naive and aware datetimes causes error
naive = datetime.now()
aware = datetime.now(timezone.utc)

# This will raise TypeError!
# result = naive < aware

# ‚úÖ Make both naive or both aware
naive_utc = datetime.utcnow()
result = naive < naive_utc  # OK, both naive
```

### Mistake 2: Wrong Format Codes

```python
from datetime import datetime

now = datetime.now()

# ‚ùå Wrong - lowercase y is 2-digit year
print(now.strftime("%y-%m-%d"))  # 24-01-15

# ‚úÖ Correct - uppercase Y is 4-digit year
print(now.strftime("%Y-%m-%d"))  # 2024-01-15
```

### Mistake 3: Mutating Instead of Replacing

```python
from datetime import datetime

#‚ùå Wrong - datetime is immutable!
now = datetime.now()
# now.year = 2025  # AttributeError!

# ‚úÖ Correct - use replace()
future = now.replace(year=2025)
print(future)
```

### Mistake 4: Not Validating Parsed Dates

```python
from datetime import datetime

# ‚ùå Wrong - no error handling
# date_str = "invalid-date"
# date_obj = datetime.strptime(date_str, "%Y-%m-%d")  # ValueError!

# ‚úÖ Correct - handle errors
def safe_parse(date_string, format_string):
    try:
        return datetime.strptime(date_string, format_string)
    except ValueError:
        return None

result = safe_parse("invalid-date", "%Y-%m-%d")
if result:
    print(result)
else:
    print("Invalid date format")
```

### Mistake 5: Using == for Date Ranges

```python
from datetime import datetime

# ‚ùå Wrong - exact match rarely works with timestamps
now = datetime.now()
target = datetime(2024, 1, 15, 14, 30, 45)

if now == target:  # Unlikely to match exactly!
    print("Match!")

# ‚úÖ Correct - use ranges or date comparison
if now.date() == target.date():  # Compare just dates
    print("Same day!")
```

## Best Practices

### 1. Always Use Timezone-Aware Datetimes

```python
from datetime import datetime, timezone

# ‚úÖ Always specify timezone for production code
now = datetime.now(timezone.utc)
```

### 2. Use ISO Format for Storage

```python
from datetime import datetime

# ‚úÖ ISO format is unambiguous and sortable
now = datetime.now()
iso_string = now.isoformat()
print(iso_string)  # 2024-01-15T14:30:45.123456

# Parse back
parsed = datetime.fromisoformat(iso_string)
```

### 3. Store Timestamps in UTC

```python
from datetime import datetime, timezone

# ‚úÖ Always store in UTC, convert for display
utc_now = datetime.now(timezone.utc)

# Store this in database
timestamp_to_store = utc_now.isoformat()

# Convert to local timezone for display
local_time = utc_now.astimezone()
print(f"Display to user: {local_time.strftime('%I:%M %p')}")
```

### 4. Use date, not datetime for Date-Only Operations

```python
from datetime import date, datetime

# ‚úÖ Use date for date-only operations
birthday = date(1990, 5, 15)  # Clearer than datetime

# ‚ùå Avoid if you don't need time
# birthday = datetime(1990, 5, 15, 0, 0, 0)
```

### 5. Document Timezone Assumptions

```python
from datetime import datetime, timezone

def schedule_meeting(meeting_time):
    """
    Schedule a meeting.
    
    Args:
        meeting_time (datetime): Meeting time in UTC
    
    Returns:
        str: Confirmation message
    """
    # Explicitly document timezone expectations
    if meeting_time.tzinfo is None:
        raise ValueError("meeting_time must be timezone-aware")
    
    return f"Meeting scheduled for {meeting_time.isoformat()}"
```

## Summary

### Key Concepts

**üìå Main Classes**
- `datetime` - date and time combined
- `date` - date only
- `time` - time only
- `timedelta` - duration between dates/times

**üìå Getting Current Date/Time**
- `datetime.now()` - current local date/time
- `date.today()` - current date
- `datetime.now(timezone.utc)` - current UTC time

**üìå Formatting**
- `strftime()` - datetime to string
- `strptime()` - string to datetime
- Use format codes like `%Y-%m-%d %H:%M:%S`

**üìå Date Arithmetic**
- Use `timedelta` for adding/subtracting time
- Subtract dates to get timedelta
- Access `.days`, `.seconds`, `.total_seconds()`

**üìå Best Practices**
- Use timezone-aware datetimes
- Store in UTC, display in local time
- Use ISO format for storage
- Handle parsing errors
